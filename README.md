# E-Commerce Funnel Analysis using Google BigQuery

## Overview
Developed an end-to-end SQL analytics project in Google BigQuery to analyze e-commerce customer behavior, funnel conversion rates, traffic source performance, and revenue metrics.

## Key Analysis
- Funnel conversion tracking across customer journey stages
- Traffic source and channel performance analysis
- Revenue, AOV, and conversion KPI calculations
- Customer journey timing analysis
- Purchase funnel drop-off identification

## Tech Stack
- Google BigQuery
- SQL
- Google Cloud Platform (GCP)

## SQL Concepts Used
- Common Table Expressions (CTEs)
- Aggregate Functions
- CASE Statements
- Conversion Rate Calculations
- Revenue KPI Analysis
- Timestamp & Funnel Analysis

## Insights
- Identified major user drop-offs between checkout and payment stages
- Organic traffic generated higher purchase conversion rates
- Revenue per buyer exceeded revenue per visitor significantly
- Funnel analysis highlighted checkout optimization opportunities
